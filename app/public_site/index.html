О проекте

FVB (Football Value Betting) — аналитическая система для прогнозирования результатов футбольных матчей. Мы используем математические модели и алгоритмы машинного обучения для поиска ценных ставок (value bets).

Методология

  • ELO-рейтинги — адаптивная система силы команд
  • Dixon-Coles — статистическая модель пуассоновского типа
  • Стекинг — комбинирование моделей для повышения точности
  • Expected Value — отбор ставок по математическому ожиданию

Дисклеймер

Проект носит аналитический и образовательный характер. Прогнозы не являются финансовой рекомендацией. Решения о ставках вы принимаете на свой риск.
